## Project: Brain Games

Brain Games is a set of 5 games for the terminal.
In each game, answer 3 questions correctly.
If you lose, you can try again.

### This is games use next tools:

| Tools  | Version |
|:------:|:-------:|
| python |  3.11   |
| poetry |  1.6.1  |
| prompt |  0.4.1  |
| flake8 |  6.1.0  |
